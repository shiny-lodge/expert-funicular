Глоссарий веб-разработки

Общие термины

  • HTML - Hypertext Markup Language, язык гипертекстовой разметки. Управляет содержимым страницы. Содержится в файлах с расширением .html.
  • CSS - Cascading Style Sheets, каскадные таблицы стилей. Язык, отвечающий за внешний вид HTML-документа. Содержится в файлах с расширением .css или в теге
